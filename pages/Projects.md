<div style="font-size: 13pt; line-height: 1.9; max-width: 940px; margin: 0 auto 26px auto;">
  Tong Lab is currently centered on a small set of core ideas, plus the agent-and-tool routing system that helps turn those ideas into an actual working lab workflow.
</div>

<div style="display:grid; grid-template-columns: repeat(auto-fit, minmax(270px, 1fr)); gap: 20px;">
  <div style="background:linear-gradient(180deg,#f8faff 0%, #ffffff 100%); border:1px solid #e7ecfb; border-radius:16px; padding:22px; box-shadow:0 8px 24px rgba(38,75,151,0.06);">
    <h3 style="margin-top:0;">Multimodal memory alignment</h3>
    <p>Align image, text, and cross-rendered representations in one shared space so memory and retrieval can work across modalities instead of plain text only.</p>
  </div>

  <div style="background:linear-gradient(180deg,#f8faff 0%, #ffffff 100%); border:1px solid #e7ecfb; border-radius:16px; padding:22px; box-shadow:0 8px 24px rgba(38,75,151,0.06);">
    <h3 style="margin-top:0;">Temporal-semantic associative memory</h3>
    <p>Treat memory as evolving associative structure shaped by time, personal experience, and subjective linkage, rather than as static stored text.</p>
  </div>

  <div style="background:linear-gradient(180deg,#f8faff 0%, #ffffff 100%); border:1px solid #e7ecfb; border-radius:16px; padding:22px; box-shadow:0 8px 24px rgba(38,75,151,0.06);">
    <h3 style="margin-top:0;">Adaptive computing graphs</h3>
    <p>Build models whose computation changes with problem difficulty, uncertainty, or complexity, instead of forcing every task through the same fixed path.</p>
  </div>

  <div style="background:linear-gradient(180deg,#f8faff 0%, #ffffff 100%); border:1px solid #e7ecfb; border-radius:16px; padding:22px; box-shadow:0 8px 24px rgba(38,75,151,0.06);">
    <h3 style="margin-top:0;">Agent / tool router</h3>
    <p>A routing layer for deciding which agent, tool, runtime, or environment should handle a task—so the lab behaves like a coordinated system rather than a pile of disconnected utilities.</p>
  </div>
</div>
